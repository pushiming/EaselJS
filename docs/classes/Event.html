Event Class

Defined in: Event:45
Module: CreateJS

事件的构造器，用于 EventDispatcher.

由于事件对象经常会被多处使用，所以不要在事件监听函数以外的地方依赖它的任何状态或属性。

currentTarget

Object

Defined in currentTarget:80

对于冒泡的事件来说，此属性指向事件的当前目标对象；对于不冒泡的事件来说，此属性始终等同于 target. 举例： 如果 childObj 的父节点为 parentObj, 然后在 childObj 上触发了一个事件，并且冒泡到 parentObj. 那么，在 prentObj 的事件监听函数中，target 是 childObj, currentTarget 为 parentObj.

Default: null

eventPhase

Number

Defined in eventPhase:92

对于冒泡的事件来说，这个属性表示当前事件处于哪个阶段：

  1. capture phase: starting from the top parent to the target
  2. at target phase: currently being dispatched from the target
  3. bubbling phase: from the target to the top parent

Default: 0
